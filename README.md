# Bupropion Tracker

Personal medication tracker for Bupropion. Tracks start date, milestones and progress as a PWA.

## Install on Android

1. Open Chrome on your Android device
2. Navigate to the GitHub Pages URL: [https://ramsterhad.github.io/bupropion-tracker/index.html](https://ramsterhad.github.io/bupropion_tracker/)
3. Tap the three-dot menu (⋮) in the top right
4. Tap **"Install app"**
5. The app will appear on your home screen and app drawer

## Features

- Tracks days since start date
- Progress bars for time to full effect and total treatment duration
- Automatic milestone timeline based on clinical data
- 4-week check-in with doctor reminder if no improvement
- Data stored locally on device (localStorage) — nothing leaves your phone
- Dark mode support

## Data & Privacy

All entered data (medication name, dose, start date, check-in result) is stored exclusively in your browser's localStorage. No data is transmitted to any server. If the app is uninstalled or browser data is cleared, all locally stored data will be permanently deleted.
